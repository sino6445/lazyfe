# lazyfe

#hey, lazy front end
Lazy front end 輕量化容易使用的前端程式碼。不須使用設定複雜的React or Vue框架，直接整合了jQuery，所以沒有學習期和學習曲線的問題.適合需要快速開發的中小型專案。
